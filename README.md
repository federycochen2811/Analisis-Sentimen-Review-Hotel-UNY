# Review Hotel UNY

## 📌 Deskripsi Proyek
Menganalisis ulasan tamu Hotel UNY untuk memahami tingkat kepuasan pelanggan. Dengan memanfaatkan teknik **Natural Language Processing (NLP)**, data teks diolah untuk menentukan kecenderungan sentimen secara otomatis.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Sastrawi, Scikit-Learn (TfidfVectorizer, train_test_split, MultinomialNB, accuracy_score, classification_report, confusion_matrix), WordCloud, Matplotlib, Seaborn.
- **Environment:** Google Colab / Jupyter Notebook

## 📊 Tahapan Analisis
1. **Import Data**
2. **Feature Selection**
3. **Cleaning**
4. **Cleansing**
5. **Tokenisasi**
6. **Normalisasi**
7. **Stemming/Lemma**
8. **TF-IDF Vectorization**
9. **Word Frequency**
10. **Wordcloud**
11. **Modeling**
12. **Evaluation**

## 📈 Hasil
- Akurasi Model training: **89.29%**
- Akurasi Model testing: **82.76%**
- Pengunjung Hotel UNY mayoritas memberikan ulasan positif. Model memiliki F1-Score 0.90 untuk kelas positif, yang berarti sangat akurat dalam mengenali kepuasan pelanggan.
